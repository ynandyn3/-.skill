周日采购清单

为这 7 天准备

菠菜、小白菜、生菜、鲜鱼虾放前半周吃；带馅的小包子、小馄饨本身算一道菜，再配一道汤或素菜就够。

周日备菜提醒

  • 肉类按 1 餐 1 小袋分装冷冻：牛肉碎、鸡腿丁、猪肉末各分 2-3 袋，主要给午餐；早餐想吃肉时再取一小袋。
  • 南瓜、胡萝卜可以提前蒸到半熟，冷藏 2-3 天内优先用。
  • 西兰花、花菜、豆角、西葫芦先洗净控干；叶菜不建议提前切碎，做前再处理。
  • 鱼虾如果买鲜的，周一周二优先吃；也可以直接买小份冷冻鳕鱼/虾仁。
  • 给老人留一张“今日菜单”，只写菜名和关键处理：切小、煮软、少盐。

没买到就这样换

  • 牛肉 -> 鸡肉丁 / 猪肉末 / 鸡蛋。
  • 鳕鱼或鲈鱼 -> 虾仁 / 三文鱼 / 豆腐羹。
  • 西兰花 -> 花菜 / 菠菜 / 小白菜。
  • 芦笋、玉米笋 -> 西葫芦 / 胡萝卜条 / 豆角。
  • 面条 -> 软饭 / 小馄饨 / 小米粥。
